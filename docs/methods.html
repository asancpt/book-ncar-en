3.2 Datasets

R itself contains two pharmacokinetic datasets, called Theoph and Indometh. In this tutorial we will use these datasets. (Figure 6.3) In NCA R packages, the input dataset should be prepared as a long format as Theoph dataset shown in Table 3.1, and if a dataset is in a wide format, it can be easily converted using gather() function in tidyr package. Data preparation is one of the most important parts in accurate NCA process, therefore using trackable and reproducible code scripts like R, SAS, or python is highly recommended. Using R for data preparation and NCA together will significantly reduce time and efforts for performing NCA.
